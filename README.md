# Decision Tree Text Classifier

A trainable text classification system implemented in Java using a binary decision tree.

## Overview

This project builds a machine learning–style text classifier from scratch. It processes raw text data, extracts features, and constructs a decision tree model for classification.

## Pipeline

- Load and parse CSV data  
- Extract features from text input  
- Train a binary decision tree model  
- Perform classification using recursive traversal  

## Features

- Recursive decision tree construction  
- Recursive prediction (inference)  
- Feature selection based on word frequency differences  
- Threshold-based splitting for classification  
- Model persistence with save/load functionality  

## Technical Highlights

- Implemented custom binary tree structure  
- Used recursion for both training and prediction  
- Designed end-to-end data pipeline (data → features → model → prediction)  
- Applied data-driven decision logic for classification  

## Tech Stack

- Java  
- Binary Tree  
- Recursion  
- Data Processing  
- Algorithm Design
